## 使用

1. 📥 下载项目代码。

```bash
git clone https://github.com/longfeiyu127/games.git games-web
cd games-web
```

2. 🏴󠁶󠁥󠁷󠁿 进入目录安装依赖，国内用户推荐使用 [cnpm](https://cnpmjs.org) 进行加速。

```bash
yarn install
```

或者

```bash
npm install
```

3. 🎬 启动本地服务器。

```bash
npm run start
```

4. 👀 启动完成后打开浏览器访问 [http://localhost:3000](http://localhost:3000)，如果需要更改启动端口，可在 `.env` 文件中配置。
